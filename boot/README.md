# boot

From Windows or Unix boot the basics, so the rest can also work

## windows 


```sh

boot.ps1

deboot.ps1
```

unix boot.sh

```sh

chmod +x ./boot.sh
chmod +x ./deboot.sh

./boot.sh

#./deboot.sh

```

## Unix

unix deboot.sh

```sh

chmod +x ./boot.sh
chmod +x ./deboot.sh

./deboot.sh

```
